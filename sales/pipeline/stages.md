# BossAI Sales Pipeline

> 目标：从 0 → 100 个付费用户
> 每个阶段定义、行动、指标、转化率目标

---

## Pipeline 阶段

```
Lead → Contacted → Demo Scheduled → Trial → Activated → Paid → Renewed → Referral
```

---

## 阶段详解

### 1. Lead（线索）

**定义：** 潜在客户首次被发现。
**来源：** 小红书评论 / 微信群 / 朋友推荐 / 官网下载 / 社媒私信
**行动：** 记录到 Lead Database，24 小时内首次联系。
**目标：** 每周新增 50 个 Lead。
**下一阶段转化率目标：** 60% → Contacted

### 2. Contacted（已联系）

**定义：** 已与潜在客户建立沟通。
**方式：** 微信私信 / 群聊 @ / 邮件
**行动：** 了解需求、确认兴趣、发送产品介绍。
**目标：** 回复率 > 40%。
**下一阶段转化率目标：** 30% → Demo Scheduled

### 3. Demo Scheduled（已约演示）

**定义：** 约定时间进行产品演示。
**方式：** 腾讯会议 / 微信通话 / 录播视频
**行动：** 根据客户行业准备演示场景。
**目标：** 每周完成 5 场 Demo。
**下一阶段转化率目标：** 70% → Trial

### 4. Trial（试用中）

**定义：** 客户已下载并开始试用。
**行动：** 24 小时内发送欢迎消息，48 小时跟进首次使用。
**目标：** 试用转化率 > 50%。
**下一阶段转化率目标：** 40% → Activated

### 5. Activated（已激活）

**定义：** 客户已输入激活码。
**行动：** 确认激活成功，发送使用指南，收集反馈。
**目标：** 激活后 7 天内完成首次完整创作。
**下一阶段转化率目标：** 80% → Paid

### 6. Paid（已付费）

**定义：** 客户已完成付款。
**行动：** 发送发票、确认 License、加入 VIP 群。
**目标：** 当月确认收入。
**下一阶段转化率目标：** 90% → Renewed

### 7. Renewed（已续费）

**定义：** 客户持续使用并推荐他人。
**行动：** 定期回访、收集案例、邀请参与产品讨论。
**目标：** V1.x 续费率 > 90%（一次性买断，续费指推荐/升级）。
**下一阶段转化率目标：** 20% → Referral

### 8. Referral（推荐）

**定义：** 客户推荐新用户。
**行动：** 确认推荐来源，发放推荐奖励。
**目标：** 每个活跃客户平均推荐 1 个新客户。
