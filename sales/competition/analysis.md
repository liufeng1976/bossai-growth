# BossAI Competition Analysis

> 对比主流 AI 工具，明确 BossAI 差异化优势

---

## 直接竞品

### ChatGPT Plus（$20/月）

| 维度 | ChatGPT Plus | BossAI Desktop |
|------|-------------|---------------|
| 定位 | 通用 AI 聊天 | AI 内容工作台 |
| 内容类型 | 文本 | 文本+图片+视频+音频 |
| 多步骤工作流 | 无（手动切换） | Director 自动规划执行 |
| Provider 选择 | 仅 OpenAI | 25+ Provider 可选 |
| 数据存储 | 云端 | 本地 |
| 价格 | $20/月 → $240/年 | ¥299 一次性 |
| 离线使用 | 不支持 | 本地软件（API 需联网） |

**BossAI 优势：** 内容工作流自动化、多 Provider、本地数据、永续授权。

### Gemini（免费 / $20/月）

| 维度 | Gemini | BossAI Desktop |
|------|--------|---------------|
| 定位 | 通用 AI 助手 | 内容创作工作台 |
| 内容创作 | 仅文本 | 多模态内容包 |
| 工作流 | 无 | Director 自动规划 |
| Provider | 仅 Google | 25+ 可选 |

**BossAI 优势：** 专注内容创作场景、自动化工作流。

### Claude（$20/月）

| 维度 | Claude | BossAI Desktop |
|------|--------|---------------|
| 定位 | 通用 AI | 内容工作台 |
| 内容类型 | 文本为主 | 文+图+视频+音频 |
| 场景 | 通用对话 | 内容创作垂直场景 |

**BossAI 优势：** 支持 Claude 作为 Provider + 内容创作垂直能力。

---

## BossAI 核心差异化

| 差异化点 | 竞品做不到的 |
|---------|------------|
| **Director 工作流** | 输入一句话 → 自动规划多步骤 → 调用不同 Provider → 全部保存 |
| **Provider 中立** | 不锁定任何模型厂商，用户自由选择 |
| **本地优先** | 数据不上传云端 |
| **一次性买断** | 不用按月付费 |
| **Capability Bus** | 统一架构，未来兼容 MCP/HTTP/Workflow |
