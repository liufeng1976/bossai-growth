# BossAI Trial Funnel

> 从官网下载到付费的完整用户旅程

---

## Funnel 阶段

```
官网下载 → 安装 → Welcome Wizard → 配置 AI Connections → 第一次生成 → License 激活 → 付费
```

---

## 阶段 1：官网下载

**用户行为：** 访问官网，点击下载按钮，下载安装包。
**成功标准：** 下载完成。
**流失原因：** 官网没说明白产品价值、下载按钮不明显、安装包太大。
**优化建议：** Home 页 Hero 区直接展示价值，"下载" 按钮始终可见，安装包体积优化。
**预期转化率：** 访问 → 下载 30%。

## 阶段 2：安装

**用户行为：** 运行安装程序，完成安装。
**成功标准：** 安装成功，首次启动。
**流失原因：** 安全软件拦截、系统不兼容、安装包损坏。
**优化建议：** 提供安装教程、SHA256 校验码、常见安装问题 FAQ。
**预期转化率：** 下载 → 安装 80%。

## 阶段 3：Welcome Wizard

**用户行为：** 首次启动 → 语言选择 → API 配置。
**成功标准：** 完成 Wizard，到达首页。
**流失原因：** 没有 API Key、配置过程太复杂、不知道该填什么。
**优化建议：** Wizard 提供 "跳过" 按钮（使用 Mock 模式）、提供 DeepSeek 注册引导、显示 10 次免费试用提示。
**预期转化率：** 安装 → Wizard 完成 60%。

## 阶段 4：配置 AI Connections

**用户行为：** 进入 API 配置页，添加 API Key。
**成功标准：** 至少配置一个 Provider 并测试通过。
**流失原因：** 不知道去哪申请 API Key、配置后测试失败、没有 DeepSeek 账号。
**优化建议：** 内置 DeepSeek 注册引导链接、提供默认配置模板、简化测试流程。
**预期转化率：** Wizard 完成 → API 配置 70%。

## 阶段 5：第一次生成

**用户行为：** 在首页输入任务或使用 Director，完成第一次 AI 生成。
**成功标准：** 成功生成内容，看到结果。
**流失原因：** API Key 无效、网络问题、不知道输入什么。
**优化建议：** 首页默认显示示例 Prompt、提供 Demo 模式一键体验、API 失败时给出明确错误提示。
**预期转化率：** API 配置 → 首次生成 80%。

## 阶段 6：License 激活

**用户行为：** 试用次数用完后看到激活提示，输入激活码。
**成功标准：** 激活成功。
**流失原因：** 不知道要付费、价格不清晰、购买流程复杂。
**优化建议：** 试用期结束前 3 次发送提醒、内置购买引导、支持微信/支付宝。
**预期转化率：** 首次生成 → 激活 30%。

## 阶段 7：付费

**用户行为：** 完成付款。
**成功标准：** 收到款项，License 生效。
**流失原因：** 支付方式不支持、价格太高、不确定是否值得。
**优化建议：** 提供试用转付费无缝体验、Early Access 折扣、满意保证。
**预期转化率：** 激活 → 付费 80%。

## 流失原因优先级

| 优先级 | 流失阶段 | 原因 | 缓解措施 |
|--------|---------|------|---------|
| P0 | API 配置 | 用户没有 API Key | 内置 DeepSeek 注册引导 |
| P0 | 首次生成 | 不知道输入什么 | 默认示例 Prompt + Demo 模式 |
| P1 | Wizard | 配置太复杂 | 支持跳过 + Mock 模式 |
| P1 | 激活 | 价格不清晰 | 内置明确价格展示 |
