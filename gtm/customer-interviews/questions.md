# BossAI Customer Interview Questions

> 用于采访第一批客户，了解真实使用体验

---

## 分类一：安装体验（3 问）

1. 你是从哪里知道 BossAI 的？
2. 下载和安装过程顺利吗？遇到什么问题？
3. 第一次打开软件时的 Welcome Wizard 体验如何？

## 分类二：API 配置（3 问）

4. 配置 API Key 的过程顺利吗？
5. 你用了哪个 Provider？为什么选它？
6. 申请 API Key 的过程中有遇到困难吗？

## 分类三：功能使用（5 问）

7. 你第一次用 BossAI 做了什么？
8. Director 功能用了吗？体验如何？
9. AI Chat、Image、Video 用了吗？哪个最有用？
10. Asset Library 功能有用吗？有什么改进建议？
11. 有没有哪个功能让你觉得特别有价值？

## 分类四：价格（3 问）

12. 你觉得 ¥699 的价格合理吗？
13. 你更倾向于一次性买断还是按月订阅？
14. 如果让你定价，你觉得多少钱合适？

## 分类五：价值感知（3 问）

15. BossAI 帮你解决了什么问题？
16. 使用 BossAI 后，你的内容创作效率提升了多少？
17. 你觉得 BossAI 和 ChatGPT 相比最大的区别是什么？

## 分类六：改进建议（3 问）

18. 你最希望 BossAI 增加什么功能？
19. 有什么功能是你觉得没用或者可以去掉的？
20. 如果让你给 BossAI 提一个最重要的建议，会是什么？

---

## 采访流程

1. 开场：说明采访目的（5 分钟）
2. 安装体验（5 分钟）
3. API 配置（5 分钟）
4. 功能使用（10 分钟）
5. 价格（5 分钟）
6. 价值感知（5 分钟）
7. 改进建议（5 分钟）
8. 收尾：感谢 + 赠送小礼物

总计：约 40 分钟
