# BossAI Growth

BossAI Growth 是 BossAI 的商业运营仓库。不开发软件，不修改 Desktop 代码，不新增 AI 能力。唯一 KPI 是帮助 BossAI Desktop 获得第一批真实付费用户。

BossAI 是 AI Work Operating Platform。用户使用自己的 AI API Key（BYO AI），一句话完成文案、图片、视频、音频创作。BossAI 不提供模型，不转售 Token。Desktop 一次买断，本地运行。
