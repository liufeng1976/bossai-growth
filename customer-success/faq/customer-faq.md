# BossAI Customer Success FAQ

> 面向已购买或试用客户的常见问题

---

## 安装与配置

### 安装后无法启动怎么办？
确保系统为 Windows 10 64 位以上。尝试以管理员身份运行。如果提示缺少 DLL，安装 Visual C++ Redistributable。

### API Key 在哪里配置？
打开软件后，点击导航栏的「API」按钮进入 API Center。选择 Provider 类型，点击「配置」输入 API Key。

### DeepSeek API Key 怎么获取？
访问 platform.deepseek.com，注册账号，在 API Keys 页面创建。

## 使用问题

### Director 执行失败了怎么办？
检查 API Center 中对应 Provider 的状态是否 🟢 Ready。确认 API Key 有效且账户有足够余额。

### 视频生成一直在等待？
视频生成通常需要 1-5 分钟。如果超过 10 分钟，在 API Center 检查视频 Provider 的状态。

### 生成的内容保存在哪里？
所有内容自动保存在 Asset Library（导航栏「资产」按钮）。按类型分类，可随时查看和管理。

### 可以导出生成的内容吗？
目前可以在 Asset Library 查看内容。视频导出功能正在完善中。

## 授权

### 激活码可以退吗？
Early Access 期间提供满意保证。如有问题请联系我们。

### 换电脑了怎么办？
当前版本为设备锁定。联系客服协助迁移，提供购买凭证。

### 激活码丢失了？
联系客服，提供购买凭证可重新获取。

## 计费

### 除了买 BossAI，还有哪些费用？
你需要自己承担 AI Provider 的 API 调用费用。DeepSeek、OpenAI 等按使用量收费，BossAI 不收取任何 API 中间费用。

### 每月 API 费用大概多少？
取决于使用量。轻度用户每月约 ¥20-50（DeepSeek），重度用户约 ¥100-300。

### V1.1 需要重新购买吗？
不需要。V1.x 全部免费更新。
