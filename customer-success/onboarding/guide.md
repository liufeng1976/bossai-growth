# BossAI Customer Onboarding Guide

> 目标：新用户在 24 小时内完成第一次付费生成

---

## 第 1 步：欢迎（用户下载后立即发送）

**渠道：** 微信 / 邮件

**内容：**
```
欢迎使用 BossAI Desktop V1.0 Early Access 🎉

24 小时内完成第一次创作，你将感受到 AI 内容团队的完整工作流。

快速开始：
1. 安装后打开软件
2. 按照 Welcome Wizard 配置 DeepSeek API Key
3. 在首页输入：帮我制作一条宠物保健品短视频
4. 观看 Director 自动完成

有任何问题直接回复这条消息，我随时在线。
```

---

## 第 2 步：首次使用引导（用户安装后 2 小时）

**检查：** 用户是否已完成安装？
**渠道：** 微信

**内容：**
```
安装顺利吗？第一次使用建议试试以下 3 个功能：

1. Director Mode（首页输入框）—— 一句话自动完成
2. AI Chat（导航栏）—— DeepSeek 流式对话
3. API Center（导航栏 API）—— 查看 Provider 状态

遇到问题随时问我。
```

---

## 第 3 步：首次生成跟进（用户使用后 24 小时）

**检查：** 用户是否生成了内容？
**渠道：** 微信

**内容：**
```
用 BossAI 做了第一个内容了吗？感觉怎么样？

如果还没开始，试试这个：
输入：帮我制作一条 [你的行业] 的推广短视频

另外提醒：新用户有 10 次免费试用，用完再决定是否激活。
```

---

## 第 4 步：激活转化（用户使用 3-5 次后）

**检查：** 用户是否接近试用上限？
**渠道：** 微信

**内容：**
```
看你的试用次数快用完了，用得还顺手吗？

如果觉得有用，现在激活有 Early Access 优惠价。
激活后永久使用，V1.x 全部免费更新。

需要激活码随时说。
```

---

## 第 5 步：长期维护（激活后）

**频率：** 每周一次
**渠道：** 微信

**内容：**
```
这周用 BossAI 做了什么内容？
有没有遇到什么问题？

另外想问问你，最希望 V1.1 增加什么功能？
```

---

## Onboarding Checklist

对每个新客户执行：

- [ ] 发送欢迎消息（下载后立即）
- [ ] 确认安装成功（2 小时后）
- [ ] 确认首次生成（24 小时内）
- [ ] 收集首次使用反馈（3 天内）
- [ ] 发送激活引导（接近试用上限时）
- [ ] 确认激活
- [ ] 每周回访
