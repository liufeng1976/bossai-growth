# Getting Started — 申请你的第一个 AI 密钥

> P0 · 5 分钟图文教程

---

## 入口

在 AI Connections Wizard 中点击「没有 AI 密钥？」按钮。

## 页面结构

标题：获取你的 AI 密钥
说明：选择你想使用的 AI 服务，按照以下步骤申请。平均只需 5 分钟。

## DeepSeek（推荐）

**为什么选 DeepSeek：** 性价比高，注册即送 500 万 Token，国内直连速度快。

**步骤：**
1. 访问 platform.deepseek.com
2. 点击「注册」，使用邮箱或手机号注册
3. 登录后进入「API Keys」页面
4. 点击「创建 API Key」
5. 复制密钥（以 sk- 开头）
6. 回到 BossAI，粘贴密钥

**预计时间：** 3 分钟
**费用：** 注册送 500 万 Token，足够使用数月

## OpenAI

**步骤：**
1. 访问 platform.openai.com
2. 注册账号（需要海外手机号验证）
3. 登录后进入「API Keys」页面
4. 点击「Create new secret key」
5. 复制密钥（以 sk-proj- 开头）
6. 回到 BossAI，粘贴密钥

**预计时间：** 5 分钟
**费用：** 按量计费，约 $0.15/百万 Token

## Claude

**步骤：**
1. 访问 console.anthropic.com
2. 注册账号
3. 登录后进入「API Keys」页面
4. 点击「Create Key」
5. 复制密钥（以 sk-ant- 开头）
6. 回到 BossAI，粘贴密钥

**预计时间：** 5 分钟

## Gemini

**步骤：**
1. 访问 aistudio.google.com
2. 使用 Google 账号登录
3. 进入「API Keys」页面
4. 点击「Create API Key」
5. 复制密钥（以 AIzaSy 开头）
6. 回到 BossAI，粘贴密钥

**预计时间：** 3 分钟

## 常见问题

**Q: 申请 API Key 需要付费吗？**
A: DeepSeek 注册送免费额度。OpenAI 需要绑定支付方式但新用户有 $5 免费额度。

**Q: 哪家最便宜？**
A: DeepSeek 性价比最高，国内用户推荐首选。

**Q: API Key 安全吗？**
A: API Key 仅保存在你的电脑本地。BossAI 不会上传你的密钥。
