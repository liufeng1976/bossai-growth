# 首次生成体验设计

> P1 · 让用户一键完成第一次生成

---

## 当前问题

用户完成 API 配置后进入首页，看到一个输入框，但不知道要输入什么。

## 解决方案

### 首页推荐任务

API 配置完成后，首页显示推荐任务卡片：

```
今天试试：
┌─────────────────────────────────────┐
│ 🐾 生成一条宠物保健品广告            │
│ 📝 写一篇小红书种草笔记              │
│ 🎬 制作一条 TikTok 短视频            │
│ 📧 生成一封营销邮件                  │
└─────────────────────────────────────┘
```

点击任意任务自动填入输入框并开始执行。

### 任务内容

| 任务 | Prompt |
|------|--------|
| 宠物保健品广告 | 帮我制作一条宠物保健品短视频 |
| 小红书笔记 | 写一篇小红书种草笔记，推荐宠物保健品 |
| TikTok 视频 | 制作一条 15 秒 TikTok 广告，推广宠物保健品 |
| 营销邮件 | 写一封宠物保健品新品发布邮件 |

### 一键体验

点击任务卡片后：
1. 自动填入输入框
2. 2 秒后自动点击「开始执行」
3. 进入 Director 执行流程
4. 用户看到的是完整执行过程

### 执行中

显示：
- BossAI 正在工作中...
- 当前执行步骤（写文案 → 出图 → 做视频）
- 进度条
