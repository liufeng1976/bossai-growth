# Experience Mode — 无 API Key 体验设计

> P0 · 让不配置 API 的用户也能看到 BossAI 价值

---

## 触发条件

用户在 AI Connections Wizard 中点击「先看看 BossAI 能做什么」

## 体验内容

### Demo Project

首次进入时自动创建一个 Demo Project，包含：
- "宠物保健品品牌" 项目
- 预设品牌名称和语气
- 示例产品列表

### Demo Director

首页显示一条预设演示任务，用户一键即可观看 Director 执行全过程（动画模拟）：
"帮我制作一条宠物保健品短视频"

### Demo Assets

Asset Library 中预置：
- 1 条 AI Chat 对话记录
- 1 张产品图
- 1 条视频（Demo）
- 展示 Asset Library 界面

### Demo Chat

AI Chat 页面显示预设对话示例，用户可以查看但不能发送真实消息（因为没有 API Key）。

### 限制提示

Experience Mode 中所有操作都显示：
"这是演示内容。配置 AI 密钥后可生成真实内容。"

### 退出入口

每个页面都提供「连接你的 AI」按钮，点击后回到 AI Connections Wizard。

### AI Connections 入口

导航栏显示：⚡ 连接你的 AI（高亮提示）
点击后回到 AI Connections Wizard。

### 体验时长

不限制体验时长。用户可在 Demo 模式中停留任意时间，随时可以配置 API 转正。
