# BossAI Desktop V1.0 Early Access

## 产品简介

BossAI Desktop 是一款 AI 原生内容工作台。使用你自己的 API Key，一句话即可完成聊天、图片、视频、音频创作。所有数据保存在本地，无需云依赖，无需订阅。

## 核心能力

### Director 智能工作流
输入自然语言任务，BossAI 自动规划执行步骤，调用对应 Provider，按顺序执行，自动保存所有结果。用户不需要手动切换任何工具。

### 双模式设计
Professional Mode：直接进入 AI Chat / Image / Video / Audio，适合有经验的用户。
Director Mode：一句话描述需求，AI 自动完成整个工作流。

### 统一 Provider 管理
API Center 一站式管理所有 AI Provider。支持 25+ Provider，覆盖 LLM、图片、视频、音频。

### 本地资产库
所有生成内容自动保存到本地，按项目分类，不丢失任何创作记录。

### 项目记忆
BossAI 记住你的品牌、产品、语调和偏好。每次工作不需要重复配置。

## 系统架构

Business Context → Director → Capability Bus → Provider Router → AI Providers → Asset Library
