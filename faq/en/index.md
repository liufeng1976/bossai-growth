# BossAI Desktop FAQ

## Product

### 1. What is BossAI Desktop?
BossAI Desktop is an AI-native content workspace. Use your own API keys to complete chat, image, video, and audio creation with one sentence. All data stays local. No cloud dependency.

### 2. How is it different from ChatGPT?
ChatGPT is a chat interface. BossAI is a content workspace — multi-step automated workflows (Director), unified multi-provider management, local asset library, project memory. Complete copy, images, video, and voice in one tool.

### 3. Is BossAI a SaaS?
No. BossAI Desktop is a local desktop app. All data stays on your machine. Optional SaaS sync coming in V2.

### 4. Does it require internet?
Yes. BossAI runs locally but AI generation requires cloud AI Provider APIs.

### 5. Mac support?
Windows 10+ only. Mac version on the roadmap.

### 6. Mobile support?
V1 is Windows Desktop only. Mobile in V2 roadmap.

## Installation

### 7. How to install?
Download the installer, run it, follow the setup wizard.

### 8. System requirements?
Windows 10+ (64-bit), 4GB RAM, 500MB disk space.

### 9. Installation blocked by antivirus?
Select "Run anyway" or add to whitelist.

### 10. How to update?
V1.x updates are free. Download and install the latest version.

## API

### 11. What API keys do I need?
At minimum, one LLM Provider API key. DeepSeek recommended.

### 12. Are API keys safe?
All keys are stored in your local machine storage only.

### 13. Chinese provider support?
Yes. Qwen, Kimi, GLM, MiniMax, Jimeng, Kling, Doubao all supported.
