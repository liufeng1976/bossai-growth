# BossAI FAQ

## Installation

1. **BossAI supports which OS?** Windows 10/11 64-bit for V1.  
2. **Is macOS supported?** Not in V1; reserved for future versions.  
3. **Can I install offline?** Yes, after downloading the installer.  
4. **Why does Windows show a warning?** Early Access builds may not have reputation yet.  
5. **Can I change install path?** Yes, NSIS installer supports custom path.  
6. **How do I uninstall?** Use Windows Apps settings or the uninstaller.  
7. **Will uninstall delete my works?** User data may remain in app data unless manually removed.  
8. **Can I install on multiple PCs?** Early Access policy is one user; commercial policy to confirm.  

## API

9. **Do I need an API key?** BossAI V1 uses BYOK for real provider calls.  
10. **What is BYOK?** Bring Your Own Key. You use your own model/API account.  
11. **Can I use BossAI without API keys?** Yes, Mock mode supports demo and fallback.  
12. **Which LLM is supported?** DeepSeek is the main early provider; more are reserved.  
13. **Which video providers are supported?** Mock, MiniMax, Kling, Runway structures are prepared.  
14. **Can I use OpenAI?** Reserved for provider expansion.  
15. **Where are API keys stored?** Locally on the user device in V1.  
16. **Can BossAI pay API costs for me?** Not in V1 BYOK pricing.  

## Pricing

17. **What is Starter?** Entry plan for individual validation.  
18. **What is Professional?** Main plan for business owners and creators.  
19. **What is Business?** For teams, agencies, or multiple brands.  
20. **What is Enterprise?** Reserved custom package.  
21. **Does price include model cost?** No, V1 pricing is BYOK.  
22. **Is there a free trial?** Early Access can provide limited demo experience.  
23. **Can I get a refund?** Refund rules should be confirmed before purchase.  
24. **Will SaaS pricing be different?** Yes, Cloud pricing will be separate.  

## License

25. **How is BossAI activated?** With an activation code.  
26. **Can I transfer license?** Early Access transfer policy is manual.  
27. **What if activation fails?** Contact support with screenshot and device info.  
28. **Does BossAI require login?** V1 Desktop does not require cloud login.  
29. **Can license expire?** Annual plans can expire depending on purchase agreement.  
30. **Can I use one key for team members?** Not unless explicitly granted.  

## Privacy and Data

31. **Where are works saved?** Locally on the computer.  
32. **Does BossAI upload my files?** Only provider API calls use external services when configured.  
33. **Are generated videos local?** Mock videos and exports are local.  
34. **Are API keys encrypted?** V1 should be treated as local-only storage; do not share devices.  
35. **Can I delete data?** You can remove local app data manually.  
36. **Does BossAI train models with my data?** BossAI does not train models in V1.  

## Models and Providers

37. **Why Provider architecture?** BossAI should not be locked to one model vendor.  
38. **What happens if a provider fails?** BossAI falls back to Mock/local demo flow.  
39. **Can I switch provider?** Yes, provider settings are prepared.  
40. **Which provider is recommended for Chinese users?** DeepSeek for text; MiniMax/Kling are future video priorities.  
41. **Does BossAI include digital human training?** No.  
42. **Does BossAI auto-publish?** No.  

## Product Usage

43. **What does one generation include?** Title, script, cover copy, subtitles, social copy and video package.  
44. **Can I edit the result?** Director mode supports review/refine flows.  
45. **Can I export video?** Yes.  
46. **Can I save works?** Yes, works are saved locally.  
47. **Can I create templates?** Template center is prepared for sales/demo positioning.  
48. **Can I use it for clients?** Business plan is recommended for client work.  

## Updates and Support

49. **How are updates delivered?** Manual download in V1.  
50. **Is auto-update supported?** Reserved.  
51. **How do I report bugs?** Send reproduction steps, screenshot, OS version and installer version.  
52. **What is the roadmap?** Desktop first, Cloud and Mobile later after validation.
