常见问题

使用自己的 API Key。本地运行。不绑定任何厂商。

BossAI 和 ChatGPT 有什么区别？

ChatGPT 是聊天界面。BossAI 是内容工作台。输入一句话，自动完成文案、配图、视频。

需要自己买 API Key 吗？

是的。BossAI 使用你自己的 API Key。我们不提供模型，不赚 Token 差价。DeepSeek 注册送 500 万 Token。

数据安全吗？

所有数据保存在你电脑本地。API Key 仅存储在本地。BossAI 不上传你的数据或密钥。

可以试用吗？

可以。内置 10 次免费试用，无需激活码，无功能限制。满意后购买激活码。

买断后还要付费？

V1.x 更新免费。V2 升级需单独购买。此外只需支付 AI Provider 的 API 费用。

DeepSeek、OpenAI、Claude、Gemini、通义千问（LLM）；GPT Image、即梦（图片）；可灵、Runway、Pika（视频）。
